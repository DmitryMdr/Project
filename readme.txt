В этом файле приведена вся структура сайта
Каждая папка будет обозначена так (/название папки)

/configs
	db.php - подключение БД
	settings.php - различные настройки сайта
/files - файлы для сайта (фото)
/pages - все страницы сайта
	/functions - функции
		delete_ad.php - удаление объявления
		get_ads.php - получние объявления
		upload.php - скрипт загрузки картинки и данных
		edit_profile.php - редактирование профиля
		get_user_public.php - получение публичной информации о пользователе
		upload_user_photo.php - загрузка фотографий в профиль пользователя

	ads_page.php - страница просмотра объявления
	auth.php - авторизация
	chat.php - тело чата
	edit_ad.php - редактирование объявления
	edit_user.php - редактирование пользователя
	exit.php - выход
	messages.php - альтернативная страница для отправки сообщений
	registr.php - регистрация
	newad.php - добавить новое объявление
	private_mess.php - личные сообщения пользователя
	privateoffice.php - личный кабинет
	search_ad.php - поиск объявлений
	user_public.php - страница с публичной инфой о пользователе

/parts - различные куски сайта
	footer.php - футер
	header.php - хедер
index.php
readme.txt
